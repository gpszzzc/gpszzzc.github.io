Over the past couple of decades, however, downtown Los Angeles has undergone^1 a major revival, thanks in part to[^2] the introduction[^3] of the Adaptive Reuse Ordinance[^4] in 1999.

[^2]: thanks to:由于，归功于
due to: 用于较庄重的书面语中，侧重“起因于”，在句中多作表语，有时作状语。
owing to:可以和due to换用，但在句中多作状语，也可作表语。
because of:着重某种原因的理由，在句中通常作状语。
thanks to:突出一种感激之情，含“多亏”意味。

[^3]: introduction:n. 介绍, 引见,引言, 导论,引进, 传入; 采用,新采用的东西

[^4]: Ordinance: /‘ɔrdnəns/ n. <正>条例,法令

Welcome to MWeb

What is MWeb? MWeb is a Pro Markdown writing, note taking and static blog generator App. MWeb used Github Flavored Markdown syntax. Please check the MWeb official website: http://www.mweb.im introduction video, it will show you how to use MWeb. We also suggest that you check the MWeb official help document: http://www.mweb.im/help.html

Things you should know

MWeb has two Mode, External Mode and Library Mode.
In External Mode, you can edit classic text and markdown files from anywhere on your Mac. As an example, you can point MWeb to a folder on Dropbox. Library Mode design for note taking and static blog/website generator. For more info, please check the MWeb official website introduction video and help document.
