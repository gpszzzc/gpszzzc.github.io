Alfred / Spotlight 使用指定应用打开文件

当使用Alfred或者是Spotlight搜索文件的时候，希望使用自己指定的程序打开搜索到的文件 - 鼠标移到文件上，按住command键，Alfred会出现Reveal file in Finder - 直接将搜索到的文件拖拽到指定应用程序上打开
